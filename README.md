# NeuraDuo-SKANDA
FESTIKA JAWA TIMUR 2025-Bidang Lomba AREK AI-SMART ABSENSI KELAS SKANDA DENGAN PEMROGRAMAN PYTHON

#Cara Menjalankan Program
1. Pastikan komputer sudah terpasang Python3, jika belum terpasang bisa download di https://www.python.org
2. Klik dua kali pada ikon program python
3. Buka file program yang telah dibuat dengan nama smart absensi
4. Jalankan program melalui terminal/command prompt dengan perintah
5. Ikuti instruksi di layar: masukkan nama siswa, pilih status kehadiran (Hadir/Izin/Sakit/Alpa)
6. Data absensi otomatis tersimpan dalam file absensi.txt dengan format
7. Guru dapat membuka file absensi.txt untuk melihat rekap absensi harian
8. Selesai
